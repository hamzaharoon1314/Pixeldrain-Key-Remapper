# ⌨️ Pixeldrain Key Remapper

A powerful, modern userscript that lets you fully customize keyboard shortcuts on Pixeldrain with a clean UI, instant feedback, and zero hassle.

---

## 🚀 Features

* 🔁 **Remap any shortcut** — change keys to whatever you prefer
* ❌ **Disable shortcuts** — remove keys you don’t use
* 🎨 **Beautiful UI** — clean, responsive, and categorized
* ⚡ **Real-time changes** — no reload required
* 🔍 **Search shortcuts** — quickly find any action
* 💾 **Persistent storage** — settings saved automatically
* 📤 **Export config** — backup your key mappings
* 📥 **Import config** — restore or share setups
* 🧠 **Conflict detection** — prevents duplicate bindings

---

## 📦 Installation

### 🟢 Install via Userscript Manager

1. Install a userscript manager:

   * Tampermonkey
   * Violentmonkey

2. Install the script from:

   * GreasyFork (recommended)
   * Or manually from this repository

```bash
https://greasyfork.org/en/scripts/571853-pixeldrain-key-remapper
```

3. Open Pixeldrain → Click the ⌨️ button (bottom-right)

---

## 🖥️ Usage

* Click the floating **⌨️ button** to open the remapper
* Click any key badge → press a new key
* Press **Esc** to cancel
* Use **Reset** to restore defaults
* Use **✕ Disable** to remove a shortcut

---

## 📂 Import / Export

You can easily back up or share your configuration:

* **Export** → downloads a `.json` file
* **Import** → loads a saved configuration

## ⚙️ How It Works

The script intercepts keyboard events and remaps them dynamically:

* Custom key → triggers original action
* Disabled key → blocked completely
* Original remapped key → prevented

All mappings are stored locally using userscript storage.

---

## 🎯 Supported Shortcuts

Includes all Pixeldrain shortcuts such as:

* Navigation (Next / Previous)
* Playback (Play, Mute, Repeat)
* Seeking controls
* File actions (Download, Copy link)
* UI toggles (Fullscreen, Panels)
